Сбор и использование информации

Приложение LUNCH PLACE для Android получает идентификатор устройства в виде Widevine MediaDrm.PROPERTY_DEVICE_UNIQUE_ID.
Приложение LUNCH PLACE для Android не сохраняет и не отправляет Widevine MediaDrm.PROPERTY_DEVICE_UNIQUE_ID куда-либо.
Приложение LUNCH PLACE для Android сохраняет Widevine MediaDrm.PROPERTY_DEVICE_UNIQUE_ID только в оперативной памяти устройства, пока пользовательский интерфейс LUNCH PLACE отображается на экране.
Приложение LUNCH PLACE для Android шифрует Widevine MediaDrm.PROPERTY_DEVICE_UNIQUE_ID с помощью необратимой хэш-функции sha256.
На этой странице значение Widevine MediaDrm.PROPERTY_DEVICE_UNIQUE_ID sha256 называется DEVICE_HASHED_UNIQUE_ID.
DEVICE_HASHED_UNIQUE_ID не передается третьим лицам.
Приложение LUNCH PLACE использует DEVICE_HASHED_UNIQUE_ID для получения информации от сервера приложения (App Cloud Firestore).
Эта информация связана с пользователем приложения. Эта информация включает в себя только следующее:
дата получения предыдущей скидки,
дата получения текущей скидки.
Приложение LUNCH PLACE для Android отображает DEVICE_HASHED_UNIQUE_ID на экране приложения в виде QR-кода.
Вы несете ответственность за нераспространение этого QR-кода.
Мы не можем гарантировать нераспространение DEVICE_HASHED_UNIQUE_ID.
