Моё резюме Командные проекты Руководства
Руководство по запуску android приложения, размещённого в виде исходного кода на GitHub.
Руководство для не разработчиков (менеджеров, дизайнеров, тестировщиков, рекрутёров или просто интересующихся) описывающее как заупустить версии android-приложения из GitHub репозитория. Руководство по git для не разработчиков.
Руководство для пользователей не знакомых с git, описывающие основные опреации c git: установка и настройка, скачивание проекта, получение изменений проекта, расшаривание изменений. Описание популярных методов разрешения конфликтов слияния в git.
Краткое описание некоторых популярных методов для разрешения конфликтов слияния (merge, rebase, squash) с перечисленим преймуществ и недостатков каждого.
